### 二、B树的实现

#### 二叉排序树的类型表述（与二叉树一样）（二叉树的链式存储）

```c
typedef struct TreeNode{
	int data;                   //数据域
	struct BiTNode *lchide, *rchild; //左、右孩子指针
}BSTNode, *BSTree;
```

#### B树的类型表述（m叉查找树）

```c
typedef struct TreeNode{
	int datas[m-1];                   //zui
	struct BiTNode *child[m];         //左、右孩子指针
}BSTNode, *BSTree;
```
